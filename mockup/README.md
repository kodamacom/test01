# DocuMind モックアップ

AI搭載文書管理システム「DocuMind」のインタラクティブなモックアップです。

## 📖 概要

このモックアップは、DocuMindのユーザーインターフェースと主要機能を視覚的に確認するために作成されました。**サーバー動作は不要**で、ブラウザで直接HTMLファイルを開いて閲覧できます。

非技術職のレビュアーでも簡単にシステムのイメージを理解できるよう、模擬データを使用してリアルな動作を再現しています。

## 🎯 目的

- プロダクトの外観と使い勝手を確認
- ステークホルダーへのデモ・プレゼンテーション
- UI/UXのフィードバック収集
- 開発前の要件確認

## 🚀 使い方

### 1. モックアップを開く

**方法1: ブラウザで直接開く**
```bash
# プロジェクトのmockupフォルダに移動
cd /path/to/project/mockup

# index.htmlをブラウザで開く
# Windowsの場合
start index.html

# macOSの場合
open index.html

# Linuxの場合
xdg-open index.html
```

**方法2: ファイルエクスプローラーから開く**
1. `mockup`フォルダを開く
2. `index.html`をダブルクリック

### 2. 画面遷移

モックアップには以下の画面が含まれています：

1. **ログイン画面** (`index.html`)
   - デモ用アカウント情報が既に入力済み
   - 「ログイン」ボタンをクリックするとダッシュボードへ

2. **ダッシュボード** (`dashboard.html`)
   - 統計情報の表示
   - 最近開いた文書
   - アクティビティフィード
   - フォルダ一覧

3. **文書一覧** (`documents.html`)
   - グリッドビュー/リストビューの切り替え
   - フィルター機能
   - 文書カードをクリックすると詳細画面へ

4. **文書詳細** (`document-detail.html`)
   - 文書プレビュー（イメージ）
   - AI要約表示
   - 基本情報・タグ
   - 共有設定
   - アクティビティ履歴

5. **AI検索** (`search.html`)
   - セマンティック検索のデモ
   - 検索結果の表示
   - 関連度スコア

## ✨ 主な機能デモ

### 1. ダッシュボード
- **統計カード**: 文書数、ストレージ使用量、アクティブユーザー、AI検索回数
- **最近の文書**: 最近開いた文書のリスト
- **アクティビティ**: チーム全体の最近の活動
- **フォルダ**: 組織内のフォルダ一覧

### 2. 文書管理
- **グリッドビュー**: カード形式で視覚的に文書を表示
- **リストビュー**: テーブル形式で詳細情報を表示
- **フィルター**: フォルダ、種類、ソートの選択
- **タグ**: 文書に付与されたタグの表示

### 3. AI機能
- **AI要約**: 文書の自動要約を表示
- **セマンティック検索**: 自然言語での文書検索
- **関連度スコア**: 検索結果の関連性を数値化
- **タグ自動生成**: AI が提案したタグ

### 4. 文書詳細
- **プレビュー**: 文書のプレビュー表示（イメージ）
- **基本情報**: ファイル形式、サイズ、作成者、作成日
- **共有設定**: 誰がアクセスできるかの表示
- **アクティビティ**: 文書の操作履歴

## 📝 注意事項

### これはモックアップです

- **実際の機能は動作しません**: ボタンをクリックするとアラートが表示されるか、別ページに遷移します
- **データはすべて模擬**: 表示されているデータはすべてダミーデータです
- **サーバー不要**: バックエンドサーバーとの通信は行いません
- **状態の保持なし**: ページをリロードすると初期状態に戻ります

### 動作要件

- **ブラウザ**:
  - Chrome（推奨）
  - Firefox
  - Safari
  - Edge
- **インターネット接続**: 不要（ローカルで動作）
- **サーバー**: 不要

## 🔄 カスタマイズ

### データの変更

`data/mock-data.js` を編集することで、表示されるデータを変更できます：

```javascript
// 例: 文書を追加
mockData.documents.push({
  id: 9,
  name: '新しい文書.pdf',
  type: 'pdf',
  size: '1.5 MB',
  createdBy: '新規ユーザー',
  createdAt: '2025-11-05',
  tags: ['新規', 'テスト'],
  folder: 'テストフォルダ',
  views: 0,
  summary: 'これは新しく追加した文書のサンプルです。'
});
```

### スタイルの変更

`css/style.css` を編集することで、外観をカスタマイズできます：

```css
/* 例: プライマリカラーを変更 */
:root {
  --primary-color: #9333EA; /* 紫色に変更 */
}
```

## 🐛 トラブルシューティング

### ページが真っ白になる
- ブラウザのコンソールでエラーを確認
- ファイルパスが正しいか確認（特にCSS、JSファイル）

### スタイルが適用されない
- `css/style.css` が正しい場所にあるか確認
- ブラウザのキャッシュをクリア（Ctrl+Shift+R / Cmd+Shift+R）

### データが表示されない
- `data/mock-data.js` が正しく読み込まれているか確認
- ブラウザのコンソールでエラーを確認
